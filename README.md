AIハイブリッド企業評価ツール
AI × Rule-based Hybrid Company Evaluation Tool for Job Hunting 
 Overview / 概要
本プロジェクトは、就職活動における企業研究を効率化するために開発した、 AIによる一次評価と人間による補正判断を組み合わせた企業評価ツールです。
AIが企業の暫定評価を自動生成,
ユーザーが必要に応じて補正,
補正値を優先して最終スコアを決定,
データ管理はExcelで行うため、操作はシンプルで直感的です。 
 Mode1：Excelテンプレート生成（初期化）
初回のみ、または台帳をリセットしたい場合に使用します。
 役割
評価用Excel（data.xlsx）を自動生成,
必要な列構造を統一,
Mode2が読み込むフォーマットを保証,
 出力列
企業名,
再評価フラグ,
業種（AI / 補正）,
安定性（AI / 補正）,
年収（AI / 補正）,
成長性（AI / 補正）,
WLB（AI / 補正）,
総合点,
判定,
企業評価コメント,
システム実行コメント,
更新日時 ,
 Mode2：企業評価（通常運用）
本ツールのメイン機能です。

 ① 評価軸の重み付け
本ツールでは、以下4つの評価軸に対して重み（重要度）を設定しています。
安定性,
年収,
成長性,
WLB,
これにより、ユーザーごとの価値観を反映した総合評価が可能になります。
 AIの評価はあくまで一般的な視点に基づく暫定評価であり、 最終的な判断はこの重み付けによって決まります。

 ② AIによる一次評価
企業名を入力すると、以下を自動生成します。
業種推定,
各評価軸（5段階）,
企業評価コメント,

 ③ 補正ロジック
ユーザーが補正値を入力した場合：
補正値があれば補正値を採用
なければAIの評価を採用

 ④ 最終評価の決定
補正後の採用値をもとに：
総合点を算出,
判定（ランク）を決定,

 ⑤ コメント生成
企業評価コメントは、
補正後の最終採用値をもとに,
ルールベースで毎回再生成,

 AIコメントの部分修正ではなく、評価結果に応じて再構築されます。

 設計思想

本ツールは以下を目的としています：
AIによる高速な一次評価,
人間による主観的補正,
Excelによる可視化と管理,
必要な行に対してのみAI評価を実行することで、処理の効率化と安定した運用を実現,

 AIに依存しすぎない意思決定支援ツール

 補足
Mode1は初回のみ使用,
Mode2が日常運用の中心,
Excelは保存して閉じてから実行,
OpenAI APIキーの設定が必要
